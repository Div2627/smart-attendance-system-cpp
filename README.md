# Smart Attendance System (C++)

## Description
A console-based application developed in C++ to manage student attendance.
The system allows marking attendance and tracking total and attended classes
for each student.

## Features
- Add student records
- Mark attendance as Present or Absent
- Display attendance report
- Menu-driven interface

## Technologies Used
- C++
- Classes and Objects
- Arrays
- Functions

## Learning Outcomes
- Understanding real-world logic implementation
- Hands-on experience with C++ OOP concepts
- Improved problem-solving skills
